# Nexautia E-commerce Template

Base privada y reutilizable para crear comercios electronicos para distintos clientes.

## Estado actual

- Esquema inicial de Supabase con 20 tablas.
- Seguridad RLS activada.
- Preparado para espanol (es-ES) y catalan (ca-ES).
- Cada cliente utilizara su propio proyecto de Supabase.

## Seguridad

Nunca se guardaran contrasenas, claves privadas ni datos personales en este repositorio.
